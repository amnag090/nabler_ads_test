# nabler_ads_test

 ## To Run Server
 ```
$ cd server
$ npm install
$ npm start
```
 Host will run on 127.0.0.1:8000 
 Database connection is already included in the .env file, It should connect automatically
 
  ## To Run Client
 ```
$ cd client
$ npm install
$ npm start
```
 Host will run on 127.0.0.1:3000 
 Database connection is already included in the .env file, It should connect automatically
